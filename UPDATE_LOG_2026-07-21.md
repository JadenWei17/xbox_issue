# 2026-07-21 更新日志

## 更新简述

- 简化小车安全策略：超声波仅保留距离遥测，移除 Arduino 与树莓派端的自动避障和超声波急停；GPIO17/D18 硬件急停仍保持最高优先级。
- 优化 Windows AI 服务：默认置信度提高到 0.60，更新猫检测模型，并增加仅限本机调用的优雅关闭接口。
- 升级网页控制台：改用页面内 WebRTC 播放，新增实时画面拍照、桌面持久化缓存与图片预览，并调整为中性灰液态玻璃界面。
- 改进运行体验：限制手柄控制日志输出频率，修正轨迹方向基准与服务停止流程，并加入树莓派 5 官方风扇满速配置脚本。

## 详细变更

### Arduino 固件

- 删除 `AVOID,ON`、`AVOID,OFF` 指令及避障模式状态。
- 超声波读数改为仅通过 `STATE` 遥测中的 `US=<cm>` 字段显示。
- 移除 30 cm 超声波急停判断；保留 GPIO17/D18 急停锁定和主动制动。
- 同步更新安全配置、状态类型和 Arduino 文档。

### Raspberry Pi 控制与部署

- 删除自动避障规划器、相关配置、串口指令和测试。
- 键盘距离指令恢复为一次性直达命令，不再自动拆分绕障路径。
- 新增 `pi_deploy/set_official_fan_max.sh`，用于配置 Raspberry Pi 5 官方风扇满速运行。

### Windows AI 与控制器

- AI 默认置信度由 0.45 提高到 0.60，并更新 `models/cat_detector/best.pt`。
- AI 服务新增仅允许本机访问的 `/api/shutdown`，前端停止时优先执行优雅关闭。
- 手柄控制数据日志限制为每秒一次，减少终端刷屏。

### Windows 网页控制台

- 将实时视频播放改为页面内 WHEP/WebRTC 连接。
- 新增 JPEG 拍照、`Desktop/RobotPhotos` 持久化存储、历史图片加载与网页预览。
- 支持通过 `ROBOT_PHOTO_DIR` 环境变量修改照片目录。
- 优化 AI 检测框颜色和置信度过滤，并在无目标时显示提示。
- 修正轨迹绘制的航向参考，重新组织控制台布局并应用中性灰液态玻璃样式。

## 兼容性说明

- Arduino 不再接受 `AVOID,ON` 和 `AVOID,OFF`。
- 超声波距离不会再触发停车、减速或绕行。
- 网页前端保存的照片会跨进程和系统重启保留，不会自动删除。
